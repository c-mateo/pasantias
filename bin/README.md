# bin

Resumen: Ejecutables y scripts que se colocan en `bin/` y se usan para tareas de administración de proyecto.

Ejemplos:
- `console.ts` - CLI helper para tareas ad hoc.
- `server.ts` - script para iniciar el servidor en entornos donde no se usa Docker.

Uso:
- Ejecutar con `node bin/console.ts` o usar los scripts npm preconfigurados según convenga.
