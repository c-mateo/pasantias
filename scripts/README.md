# scripts

Resumen: Scripts de utilidad y tareas (migraciones, seed, tareas por lotes, etc.).

Contenido:
- `start/` - tareas de arranque, jobs y kernel para Adonis.
- `bin/` - scripts ejecutables.

Recomendaciones:
- Las tareas que afectan la base de datos deben ejecutarse en entornos de desarrollo con backups o en bases de datos de prueba.
